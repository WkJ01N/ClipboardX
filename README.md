# ClipboardX

为 macOS 打造的极客级原生剪贴板管理神器。基于 SwiftData 和 SwiftUI 构建，纯本地运行，安全且强大。

简体中文 | <a href="./README_en.md">English</a>

### 🔒 极致的隐私与安全
- **屏幕共享防偷窥**：开启后，在 Zoom、腾讯会议或系统录屏时，剪贴板悬浮窗将对录屏画面“隐身”，彻底杜绝社死与隐私泄露。
- **敏感信息阅后即焚**：后台正则自动识别 API Key、身份证、银行卡等敏感数据，打上 🔒 标签并在自定义时间（如1分钟）后彻底销毁。
- **链接追踪参数净化**：复制链接时自动剔除 `utm_*`, `vd_source`, `si` 等烦人的追踪尾巴，支持全文本混排链接提取及自定义正则。
- **一键切断监听**：可通过菜单栏或全局快捷键，一键彻底暂停剪贴板读取。

### ⌨️ 硬核交互与生产力
- **打字机模式（模拟输入）**：独创纯文本打字机面板，底层调用 `CGEvent` 模拟物理键盘逐字敲击，完美绕过严苛的“禁止粘贴”网页与远程桌面环境。
- **右键加解密助手**：内置 Base64、URL 编码/解码。支持“智能解密”——一键还原加密乱码并置顶。
- **多维视图与焦点路由**：除了经典单行列表，新增**双列网格视图**。全面重写二维键盘焦点引擎，方向键（上下左右）穿梭丝滑流畅。
- **多重唤醒机制**：支持传统快捷键、**双击修饰键**（如连按两下 Option ⌥）或**长按单键**呼出面板。

### 💾 数据完全掌控
- **常用项收藏**：支持将常用文本/图片打上星标，永久保留，免疫数据库的自动过期清理机制。
- **本地导入与导出**：一键将整个剪贴板历史导出为 JSON 备份。支持“覆盖”与“合并去重”两种导入模式。
- **自定义存储路径**：你可以将数据库移动到任何位置（如 iCloud 云盘）以实现多端同步或深度定制。

## 📥 下载与安装

下载见 [Release](https://github.com/WkJ01N/ClipboardX/releases) 页面。

由于主播贫困潦倒，买不起苹果高贵的开发者账号，所以应用未经过 Apple 签名认证。你需要打开“允许任何来源”才能使用：
1. 打开系统设置，找到“**隐私与安全性**”，翻到最底部，将“*允许以下来源的应用程序*”改为“**任何来源**”。
2. **若你没有该选项**，请打开**终端 (Terminal)**，输入以下代码并回车验证密码：
   ```bash
   sudo spctl --master-disable
   ```
   再次打开设置即可看到“任何来源”选项。
3. 将下载的 `.zip` 压缩包解压，把里面的 `ClipboardX.app` 拖入“应用程序 (Applications)”文件夹，即可双击打开。

> **⚠️ 重要提示：**
> 若要使用**“打字机模式”**模拟键盘输入，请前往系统设置 -> 隐私与安全性 -> **辅助功能 (Accessibility)** 中，勾选并允许 ClipboardX 控制您的电脑。

## 🕹 快捷键指南

- **唤出 ClipboardX**：Ctrl + V (支持自定义，支持双击修饰键/长按唤醒)
- **快速粘贴**：Cmd + 1 ~ 9
- **列表导览**：上 / 下 / 左 / 右 方向键
- **快速预览图片/文件**：空格键 (Space)
- **确认粘贴选定项**：回车键 (Enter)
- **常用收藏切换**：Tab 键切换“历史/常用”面板
- **一键清空未固定项**：Cmd + Delete -> 回车
- **关闭悬浮面板**：ESC

## 🛠 技术架构

- **UI 框架**: SwiftUI (支持深色模式、流畅动画与组件化拆分)
- **数据持久化**: SwiftData (@Model & @Query，支持轻量级迁移与 SQLite 优化)
- **系统底层拦截**: AppKit (NSPanel 焦点路由拦截, sharingType 隐身屏蔽), CGEvent (硬件级键盘事件模拟)
- **多语言支持**: String Catalog (支持简体中文与英文无缝切换)
- **开源依赖**: KeyboardShortcuts
